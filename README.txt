Search-Plugin-for-CakePHP
=========================

This plugin offers functionalities for text search with accents management.

It contains a Behavior that makes a custom 'search' find type available to Models, with special options keys to filter results and highlight matches in results display, and a Lib class to handle Unicode accentuated characters.

For more information, please read the article about this plugin: http://patisserie.keensoftware.com/pages/gerer-les-accents-dans-les-recherches-textes
